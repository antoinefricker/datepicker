# DatePicker

> **Disclaimer** Work in progress

Inspired by Fluent UI [DatePicker component](https://developer.microsoft.com/en-us/fluentui#/controls/web/datepicker) for the layout.

## Features

-   [ ] Pick day
-   [ ] Pick week
-   [ ] Pick month
